## 📁 README.md для SiteGen

# 🎨 SiteGen — Генератор дизайна сайтов на GigaChat

Сервис, который по текстовому описанию создаёт готовую веб-страницу с помощью GigaChat API. Результат можно экспортировать в React-приложение.

![SiteGen Demo](https://i6.imageban.ru/out/2026/06/18/ef18dcb7956f36805e604ca0f2174f3a.jpg)
---

## ✨ Возможности

- 🤖 **Генерация сайтов** по текстовому описанию (лендинги, дашборды, многостраничные сайты)
- 🎯 **Двухфазная генерация** — определение типа сайта → создание по специализированному промпту
- 📊 **Дашборды и таблицы** — графики Chart.js, таблицы с 10+ столбцами, карточки статистики
- 🎨 **Стиль Replit** — чистый интерфейс с тёмной/светлой темой
- 📦 **Экспорт в React** — скачивание готового `.jsx` компонента
- 💾 **История генераций** — сохранение всех промптов и HTML (серверное хранение)
- 📏 **Адаптивные панели** — перетаскивание границ для изменения размеров
- ⚡ **Реальное время** — статусы генерации через Server-Sent Events (SSE)

## 🔧 Установка и запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/Artpardsur/sitegen.git
cd sitegen
```

### 2. Установи зависимости

```bash
# Бэкенд
npm install

# Фронтенд
cd client
npm install
cd ..
```

### 3. Настройка API ключа

Создай файл `.env` в корне проекта:

```env
GIGACHAT_CREDENTIALS=твой_ключ_сюда
```

**Как получить ключ:**
1. Зарегистрируйся на [developers.sber.ru](https://developers.sber.ru)
2. Создай проект → выбери «Модель GigaChat»
3. Скопируй `Authorization Key` из настроек API

### 4. Запусти сервер

```bash
# Терминал 1 — бэкенд
node server.js
```

### 5. Запусти фронтенд

```bash
# Терминал 2 — фронтенд
cd client
npm run dev
```

### 6. Открой в браузере

```
http://localhost:5173
```

## 📦 Экспорт в React

После генерации нажми кнопку **«Экспорт в React»** — скачается файл `GeneratedSite.jsx`.

Импортируй его в своё приложение:

```jsx
import GeneratedSite from './GeneratedSite';

function App() {
  return <GeneratedSite />;
}

export default App;
```

## 🛠 Технологии

| Компонент | Технология |
|-----------|------------|
| **Бэкенд** | Node.js, Express, GigaChat API |
| **Фронтенд** | React, Vite, CSS |
| **Генерация** | GigaChat-2-Pro / GigaChat-2 / GigaChat-2-Max |
| **Статусы** | Server-Sent Events (SSE) |
| **История** | JSON-файл + localStorage |
| **Графики** | Chart.js (для дашбордов) |

## 📁 Структура проекта

```
sitegen/
├── server.js              # Бэкенд (Express + GigaChat)
├── prompts/               # Системные промпты
│   ├── classifier.txt     # Определение типа сайта
│   ├── landing.txt        # Промпт для лендингов
│   └── multipage.txt      # Промпт для дашбордов
├── generated_sites/       # Сохранённые HTML-файлы
├── history.json           # История генераций
├── client/                # Фронтенд (React + Vite)
│   ├── src/
│   │   ├── App.jsx
│   │   ├── App.css
│   │   └── main.jsx
│   ├── index.html
│   ├── package.json
│   └── vite.config.js
├── .env                   # Переменные окружения
└── README.md
```
